\section{VISION}
Esta sección establece una visión a largo plazo del sistema que se construirá para abordar el negocio

ciclo de vida del desarrollo de productos.
 
\subsection{DECLARACION DE LA VISION}\label{sec:nada}
COVID-19 ESTADISTICAS. es un proyecto enfocado en el analisis y exposicion de datos relacionados a la pandemia del COVID-19. Refinamos los terminos de busqueda para aumentar el exito y la facilidad al navegar por el sitio web. Todo debera estar conectado a la base de datos de la plataforma para asi mismo poder dar un mejor manejo en la web a crear donde la informacion estara clara y concisa. Si el tiempo lo permite, el corrector ortográfico admitirá el reconocimiento de palabras / términos comunes
errores ortográficos y sugerencias de frases de actualidad relacionadas con los términos enviados.
Las mejoras pueden identificarse a medida que avanza el proyecto.


\subsubsection{Carasteristicas Principales}\label{sec:nada2}

-Inicio de sesion con correo electronico
